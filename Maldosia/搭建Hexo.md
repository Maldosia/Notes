###### 安装Git

```shell
yum -y install git
git --version
```

###### 安装python

```shell
curl https://www.python.org/ftp/python/2.7.14/Python-2.7.14.tgz -o python-2.7.14.tgz
tar -xvf python-2.7.14.tgz
./configure --prefix=/usr/local/python/2.7.14
make
make install
```
